Atenção! Siga as instruções abaixo para realizar o exercício.

Instruções

  • Fazer um fork do repositorio https://github.com/engenhariadesoftwareuepg/iniciando-com-devops
  • Pode utilizar o codeanywhere ou uma IDE
  • Realizar a integração com o Travis-ci
  • Realizar a integração de deploy do Heroku
  • Implementar o teste para 3 funções simples (verificar se é triângulo, cálculo de distância euclidiana, conversão de medidas, etc)
  • Implementar as 3 funções para passar nos testes

Trabalho em equipe

  • Desenvolver o trabalho em equipe 3 academicos
  • Cada um deve realizar pelo menos 1 função
  • Utilizar as opções de colaboração do github

Critérios

  • Cada membro deve implementar uma função
  • Cada função deve ser implementada em um branch diferente
  • As branchs devem ser mescladas a branch master
  • Deverá passar em todos os testes
